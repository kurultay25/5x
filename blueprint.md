
# Blueprint

## Overview

Create a Flutter application that wraps an existing website in a WebView, allowing it to be run as a mobile app and prepared for App Store submission.

## Current Request

### Goal
Convert the website `https://x5-233568038472.us-west1.run.app/` into an iOS application.

### Plan
1.  Add the `webview_flutter` package to the project.
2.  Replace the contents of `lib/main.dart` with a `WebView` widget that loads the specified URL.
3.  Ensure the app runs correctly.
